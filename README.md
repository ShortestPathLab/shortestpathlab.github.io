# ShortestPathLab Landing Page

## Install

Requires `npm` v10.8.1 and `NodeJS` v20.8.0. [Nodejs install guide](https://nodejs.org/en/download/package-manager)

In the terminal, run:

`npm install`

## Run dev server

In the terminal, run:

`npm run dev`

Open a browser and go to `https://localhost:4321`.
